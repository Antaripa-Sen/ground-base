# Groundbase

Zero-login, real-time crisis coordination for hospitality venues.

Groundbase enables staff to instantly create a live incident room, coordinate response, and follow AI-generated guidance — all from a browser, with no login or app required.


## Problem

Emergency response in venues is often:
- Uncoordinated  
- Slow  
- Lacking clear instructions  


## Solution

Groundbase provides:
- Instant incident rooms  
- AI-generated triage steps  
- Real-time task coordination  
- Frictionless team joining via QR/link  

Result: faster, structured, and coordinated emergency response.


## Features

- One-tap CRISIS activation  
- QR code + shareable join link  
- AI triage guidance (Gemini)  
- Live task board (real-time sync)  
- Pre-filled emergency call scripts  
- Multi-device instant updates  


## How It Works

1. Tap CRISIS  
2. Select emergency type  
3. Incident room is created instantly  
4. AI generates 3-step guidance  
5. Team joins via QR/link  
6. Tasks update live across devices  


## Tech Stack

- Frontend: React + Vite + Tailwind  
- Backend: Node.js + Express  
- Realtime: Socket.io  
- AI: Google Gemini API  
- Deployment: Vercel + Railway
